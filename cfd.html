How to Run
1
Install Python dependencies:
pip install numpy matplotlib osmnx networkx scikit-image pillow
2
Optional GPU — install CuPy for CUDA acceleration:
pip install cupy-cuda12x
3
Run the simulation script:
python cfd_simulation.py
4
Use the interactive sliders in the matplotlib window to control wind speed, direction, traffic density, and background AQI in real-time.
Architecture
Fluid Solver
Jos Stam's stable Navier-Stokes — diffuse → advect → project steps with Gauss-Seidel pressure solve
Road Network
Real OSMnx graph of Pragati Maidan (600m radius) · NetworkX shortest-path routing per vehicle
Emission Model
Each vehicle injects density at its trail per frame · Buildings act as friction barriers trapping urban-canyon smog
GPU Acceleration
Auto-detects CUDA via CuPy — upgrades to 500×500 grid. Falls back to NumPy at 200×200 on CPU.
